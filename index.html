프로젝트

GRIDA그리다

프로젝트 ‘그리다’ 는 유저를 통해 받은 이미지를 동양풍의 그림으로 재해석 하여 제공하는 서비스 입니다.
시연영상 : https://www.youtube.com/watch?v=C3OHCepTRgU
GITHUB(웹) : https://github.com/teamsixsense/webserver
GITHUB(tensorflow API) : https://github.com/teamsixsense/model_prac
프로젝트 "그리다"는 Neural Style Transfer를 사용하여 사용자의 이미지를 동양화와 유사한 화풍의 이미지로 만들어주는 서비스입니다. 사용한 모델은 tensorflowhub의 magenta/arbitrary-image-stylization-v1-256를 사용하였으며, django-ninja로 이미지 변환 API를 구축하였고 Django로 사이트를 구축하였습니다.

Travel Alone

제주도를 혼자 여행하는 사람들에게 사진 찍기 좋은 장소, 배경을 초기 설정한 태그 기반 추천 서비스

GITHUB : https://github.com/codebot053/team6
기획 및 회고 : https://www.notion.so/team-sixsense-project-4f87f664511c43f9a57196a4699fe37d
혼자 여행하는 사람들에게 제주도 여행지를 추천해주는 사이트를 개발하려고 하였습니다. 관련된 데이터는 구글과 트립어드바이저 등의 사이트에서 크롤링 하였고, 아이템 기반 협업필터링을 사용하여 유저가 최근 댓글 단 곳과 비슷한 여행지를 추천해주는 기능을 넣었습니다. 프론트엔드 쪽 완성이 덜 된게 아쉽습니다.
